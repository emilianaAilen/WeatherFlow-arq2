# Reglas para Claude Code

## Git

- **Nunca crear ramas que no se hayan pedido explícitamente.** Solo crear la rama indicada por la usuaria en el mensaje; no crear ramas auxiliares, de trabajo o de sistema sin autorización.
- Hacer commits en nombre de la usuaria: `Emiliana Ailen <emiliana.ailen@hotmail.com>`.
- La rama base para nuevas features es siempre `develop` salvo indicación contraria.
